# City Operations Analytics Dashboard | Power BI Project

## Project Overview

This project is a **Power BI dashboard solution** designed to analyze city-level business performance across trips, revenue, passengers, targets, and repeat traveler behavior.

The dashboard helps stakeholders monitor operational efficiency, understand passenger patterns, compare actual performance against targets, and identify high-performing and low-performing cities.

This project is divided into **three interactive dashboard views**:

- **Home / Operations Insights**
- **Targets Analysis**
- **Passenger Analysis**

## Business Objective

The goal of this project is to provide a clear and interactive view of business performance across multiple cities by answering questions such as:

- Which cities generate the highest revenue?
- Which cities contribute the most trips?
- Are actual trips meeting target trips?
- Are passenger targets being achieved?
- Which cities have stronger repeat passenger behavior?
- What is the distribution of new vs repeat passengers?
- Which cities need operational or customer experience improvements?

---

## Dashboard Pages

### 1. Home / City Level Operations Insights

This page gives a high-level operational summary across cities.

#### Key KPIs:
- **Total Trips:** 425.9K
- **Revenue:** 108.2M
- **Total Passengers:** 238.3K

#### Visuals Included:
- City-wise operations summary table
- Revenue generated by each city
- Total trips by city
- City trip contribution %
- Average fare amount
- Average fare per km
- Total passengers
- Average passenger rating
- City-wise fare analysis
- Revenue by city comparison chart

#### Insights from this page:
- **Jaipur** is the top city by revenue and trip contribution.
- **Kochi** and **Lucknow** also contribute significantly to total trips.
- Cities like **Mysore** and **Coimbatore** have lower trip contribution.
- Fare amount and fare per km vary significantly by city, showing pricing and route differences.
- Passenger ratings help identify cities with better customer satisfaction.

---

### 2. Targets Analysis

This page compares actual operational performance with target values.

#### Visuals Included:
- Target vs Actual Trips table
- City-wise status indicator (Above Target / Below Target)
- Target vs Actual passengers trend
- City-wise target vs actual trips chart
- City-wise target vs actual passenger ratings chart

#### Insights from this page:
- Cities such as **Mysore, Coimbatore, Kochi, and Jaipur** are performing above trip targets.
- Several cities are below target and may require demand generation or operational improvements.
- Passenger target tracking helps evaluate customer acquisition and retention performance.
- Rating comparison helps understand whether growth is aligned with service quality.

---

### 3. Passenger Analysis

This page focuses on customer behavior, especially repeat passenger trends.

#### Visuals Included:
- City-wise Repeat Passenger Rate (RPR%)
- New vs Repeat passenger distribution
- Repeated vs New passengers donut chart
- City-level repeat passenger trip frequency matrix

#### Insights from this page:
- **Surat** has the highest repeat passenger rate.
- **Mysore** has the lowest repeat passenger rate among the listed cities.
- The overall passenger base is dominated by **new passengers**, while repeat passengers form a smaller but critical segment.
- The repeat trip frequency table helps analyze customer loyalty by showing how often passengers return for additional trips.
